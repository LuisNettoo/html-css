Trabalhando com Listas

Listas Ordenadas

Usando a tag <ol> em conjunto com a tag <li> nós podemos criar listas ordenadas (ordered lists).

Exemplo:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ir para a faculdade
  5. Almoçar
  6. Voltar para casa
  7. Jantar
  8. Dormir

Listas não-Odernadas

Usando a tag <ul> em conjunto com a tag <li> nós podemos criar listas não-ordenadas (unordered lists).

Exemplo:

  • Arroz
  • Feijão
  • Macarrão
  • Molho de tomate
  • Calabresa
  • Sal
  • Pimenta do reino

Lista de Definição

Usando a tag <dl> em conjunto das tags <dt> e <dd> nós podemos criar listas de definição.

Exemplo:

HTML
Linguagem de marcação para criação do conteúdo dos sites.
CSS
Linguagem de marcação para a criação do design dos sites.
JavaScript
Lingugem de programação para a interatividade dos sites.
